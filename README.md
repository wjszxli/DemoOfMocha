## 如何去使用

首先  克隆代码  
```bash
$ git clone https://github.com/wjszxli/DemoOfMocha.git
```
第二步  安装依赖

```bash
$ cd DemoOfMocha
$ npm install
$ npm install --global mocha
```
第三步   运行测试

```bash
$ npm test
```
